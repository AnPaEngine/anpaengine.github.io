Projekt 1 - Beschreibung

Dieses Projekt zeigt meine Fähigkeiten im Bereich Webentwicklung. Es beinhaltet sowohl Frontend- als auch Backend-Elemente, die mit modernen Technologien erstellt wurden.

Ich habe HTML5, CSS3 und JavaScript verwendet, um eine benutzerfreundliche und interaktive Oberfläche zu erstellen. Die Backend-Logik basiert auf einer RESTful API, die mit Node.js und Express entwickelt wurde.

Technologien

  • HTML5
  • CSS3
  • JavaScript (ES6+)
  • Node.js & Express
  • RESTful API

Features

  • Interaktive Benutzeroberfläche
  • Responsive Design
  • API-Integration für dynamische Inhalte
  • Optimierung für alle Geräte
